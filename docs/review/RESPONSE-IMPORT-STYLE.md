# 代码规范调整：相对引用 → `package:` 绝对引用

## 改动说明

根据代码规范建议，将项目内所有 Dart 文件的**相对路径引用**（`../../foo/bar.dart`）统一改为 **`package:` 绝对路径引用**（`package:telemarketing_app/foo/bar.dart`）。

## 原因

1. **文件移动不断链**：相对引用在目录结构调整时必须逐文件修改；`package:` 引用目录层级不变，不受影响。
2. **来源清晰**：一眼看出引用来自本包而非外部包。
3. **项目规范统一**：消除混用两种风格的情况。
