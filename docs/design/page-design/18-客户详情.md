# 18 - 客户详情

> ⚠️ **版本规划说明（产品决断）**：客户管理模块（本页及 [17-客户列表](./17-客户列表.md)）为 **v1.1 版本规划**，**MVP（v1.0）不含客户管理，仅线索管理**。本文档作为 v1.1 设计预留，MVP 阶段不开发、不验收。

> **设计规范参考**：[00-TDesign-Flutter-设计规范](./00-TDesign-Flutter-设计规范.md)
> 组件库：[TDesign Flutter](https://github.com/Tencent/tdesign-flutter)
> 色彩 / 字体 / 间距 / 圆角 / 阴影 / 组件均遵循上述设计规范。

## 1. 页面概述

| 项目 | 说明 |
|------|------|
| **页面定位** | 展示单个客户的详细信息，包括基本信息、来源线索追溯以及来源线索的历史跟进记录 |
| **目标用户** | 所有已登录用户（TE / TM / TA） |
| **核心目标** | 让用户查看客户的完整信息，并可追溯到来源线索及其跟进历史 |
| **页面路由** | `/mine/customers/:id` |
| **入口** | 客户列表 → 点击客户卡片 |
| **重要说明** | 客户无独立的跟进记录表，MVP 阶段仅展示来源线索的历史跟进记录（只读）；客户不可编辑 |

| 数据特征 | 展示标签 |
|---------|---------|
| 跟进记录携带 `answerType` 或 `duration`（通话产生的跟进） | 电话跟进 |

| 字段缺失 / 未知值 | 跟进（兜底） |

> "微信跟进"等更细分标签为后续版本扩展预留，当前后端无区分依据，统一展示"跟进"。

## 4. 交互流程

### 4.1 页面加载

```
用户从客户列表点击卡片进入
  → 显示 TDNavBar + TDSkeleton 骨架屏
  → GET /api/tenant/customers/:id
  → 响应中包含客户基本信息 + 来源线索信息 + 来源线索的跟进记录
  → 数据返回后渲染各区域
  → 隐藏 TDSkeleton
```

### 4.2 点击来源线索卡片

```
用户点击来源线索卡片
  → TDesign 内置点击态反馈
  → 跳转至线索详情页 /leads/:leadId
```

### 4.3 来源线索不可用

```
页面加载时：
  → 若 leadId 为空 → 来源线索区域显示 "来源线索不可用"
  → 若线索查询返回 404 → 来源线索区域显示 "来源线索已删除"
  → 跟进历史区域：
      → 若来源线索存在 → 展示跟进记录
      → 若来源线索不存在 → 显示 "暂无跟进记录"
```

## 5. 状态设计

### 5.1 加载态

| 区域 | 表现 |
|------|------|
| 基本信息 | `TDSkeleton`：卡片骨架（5 行标签+值骨架） |
| 来源线索 | `TDSkeleton`：卡片骨架 |
| 跟进历史 | `TDSkeleton`：3 条时间线骨架（圆点骨架 + 3 行文字骨架） |

### 5.2 空态

| 场景 | 表现 |
|------|------|
| 客户详情加载成功但数据为空 | 理论上不会出现（从列表点击进入） |
| 无跟进记录 | 跟进历史区域显示 "暂无跟进记录" |
| 来源线索不可用 | 来源线索区域显示对应提示 |

### 5.3 错误态

| 场景 | 表现 |
|------|------|
| 客户详情加载失败 | `TDEmpty` 错误插图 + `TDButton(theme: TDButtonTheme.text)` "重试" |
| 客户不存在（404） | `TDEmpty` + "客户不存在" + `TDButton(theme: TDButtonTheme.text)` "返回" |
| 网络断开 | `TDToast` "网络连接失败" + 重试操作 |

### 5.4 成功态

数据正常渲染，无额外成功提示。

## 7. 边界情况与备注

| 场景 | 处理方式 |
|------|---------|
| 客户姓名为空 | 显示 "未命名客户" |
| 电话为空 | 显示 "无联系电话" |
| 公司为空 | 公司行显示 "—" |
| 转化日期为空 | 显示 "—" |
| 等级为空 | 不显示 `TDTag` 等级标签 |
| 来源线索 leadId 为空 | 来源线索区域显示 "来源线索不可用"，跟进历史显示 "暂无跟进记录" |
| 来源线索已被删除 | 来源线索区域显示 "来源线索已删除"，不可点击跳转 |
| 跟进记录内容过长 | 最多显示 3 行，超出省略号截断；点击条目可展开全文（可选，MVP 可不做） |
| 跟进记录很多 | 时间线区域支持内嵌滚动或显示前 10 条 + `TDButton(theme: TDButtonTheme.text)` "查看更多" |
| 客户不可编辑 | MVP 阶段无编辑按钮，无操作入口 |
| 拨号入口 | 本页**不含拨号入口**（MVP 客户详情仅做信息展示与来源追溯）；拨号入口为 **v1.1 规划**，届时在基本信息区电话行增加一键拨号 |
| 脱敏规则 | 姓名：保留首字，其余用 * 替代；电话：保留前 3 后 4，中间 4 位用 * |
| 页面返回 | 无状态需要保存 |
| 从不同入口进入 | 无论从客户列表还是其他入口进入，展示内容一致 |
