## 五、踩坑记录（已写入项目记忆）
1. `flutter install` 在本机会误找 `app-release.apk`（不存在）导致安装失败 → 改用 `adb install -r` 直接装 debug apk
2. 本机 adb 不在 PATH（`which adb` 为空），实际路径 `~/Library/Android/sdk/platform-tools/adb`（flutter devices 能识别设备却 which 不到）
3. `flutter run --use-application-binary | tail` 会缓冲日志看不到实时输出 → 改用 `adb logcat --pid` 直抓
4. MIUI 限制：`uiautomator dump` 无输出，无法自动模拟点击
